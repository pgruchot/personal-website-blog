  • Animate.css

    Add marvellous animations with this stylesheet.

    Animate.css is an additional stylesheet file that you can add to your website, that allows you to use pre-made animations. It's just css, so it won't impact your website load times as much as another javascript library. To animate an element, all you have to do is add a class "animated" and then another class with chosen animation. Animations include bouncing, flashing, pulsing - all that kind of stuff. Below you can see a codepen snippet with an example. As you can see we can later modify those animations with additional css or javascript, so they are highly customisable and always tailored to your needs.

  • Glitch animation

    Pure css glitch animation

    Glitch animation doesn't have to be complicated. In this case we use css animations. First of all we set up two images: one as an anchor and the second one to float around. Image shouldn't be monochromatic, because then we have no hue values to rotate. After we've selected our background it's time to add the animation. We use css property called "animation", after which we specify the animation name, animation length, repeat etc. In this scenario animation is simply called "glitch". We then specify whole animation sequence, using "@keyframes" almost like a selector. Two points which we have to specify here are 0% and 100%, but to see somekind of glitchy image we need some more breakpoints. As you can see every 10% theres a change in positioning (which simulates screen shake) and also @ 0% and 100% we have a hue value rotation (which changes colors).
